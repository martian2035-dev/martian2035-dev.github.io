# 第三章：无法逃避的抉择

草原的黎明总是伴随着清凉的风。朝阳刚刚从巴尔喀什湖的地平线升起，金色的光芒洒在湖水上，仿佛为这片土地披上了一层柔和的面纱。但对阿尔娜来说，这一天的晨光却没有任何温暖，反而像一把利刃，刺痛着她的心。

她坐在帐篷外的小木凳上，手里紧握着一块粗糙的木雕。那是她父亲去世前为她雕刻的一匹马，象征着草原上的自由。可如今，这个象征对她来说已成了奢望。她的眼前浮现出图尔汗的脸，那张总是带着冷笑的脸，让她厌恶到骨子里。可她别无选择。

帐篷里传来弟弟艾德尔的咳嗽声，打断了她的思绪。她放下木雕，走进帐篷。弟弟瘦弱的身体蜷缩在毡毯里，咳嗽声断断续续，显得格外虚弱。母亲坐在一旁，小心翼翼地给他喂药，那是用耶律萧然送来的药材熬成的汤剂。

“阿尔娜，”母亲抬起头，脸上的皱纹比以往更深了一些，“图尔汗今天会来吧？”

“是的。”阿尔娜低声答道。

母亲的眼神复杂，既有担忧也有一丝无奈。她张了张嘴，最终却什么也没说，只是低下头继续喂药。阿尔娜明白母亲的沉默意味着什么。无论她多么不情愿，这场婚事已经成了家族唯一的生路。

帐篷外传来急促的马蹄声，阿尔娜的心猛地一沉。果然，是图尔汗来了。

图尔汗的身影很快出现在帐篷外，他身穿厚重的羊毛披风，腰间挂着一把锃亮的长刀，眉眼间带着一抹不容置疑的威严。他翻身下马，走向阿尔娜，神情中透着一丝得意。

“阿尔娜，时间到了，你的答案呢？”他站在她面前，声音低沉却充满压迫感。

阿尔娜抬起头，直视着他的眼睛，努力让自己的声音保持平静，“我答应你。”

图尔汗的嘴角扬起一抹笑意，他满意地点了点头，“很好。三天后，我们的婚礼将在湖边举行，你会成为我图尔汗的女人，也是这片草原上最受尊敬的女人。”

阿尔娜没有回应。她的手握紧了衣摆，指甲几乎嵌入掌心，但她的脸上却没有任何表情。她知道，这场婚事意味着什么，她将失去自由，成为一个她厌恶的人的附属品。但她别无选择。为了母亲和弟弟，她必须妥协。

“我会派人来接你。”图尔汗说完，转身上马，扬长而去。他的手下紧随其后，留下了一片飞扬的尘土。

阿尔娜站在帐篷外，望着图尔汗离去的背影，心中充满了复杂的情绪。她的生活就像这片草原上的风，永远无法掌控自己的方向。

这一天的时间过得格外缓慢。阿尔娜几乎没有说话，只是默默地帮助母亲料理家务，照顾弟弟。母亲看出了她的心事，但却什么也没问，只是偶尔投来关切的目光。

夜幕降临，整个草原被静谧的黑暗笼罩，只剩下远处契丹军营的篝火在微微闪烁。阿尔娜走出帐篷，独自来到湖边。湖水在月光下散发出银色的光芒，波光粼粼，像一片无边无际的镜子。

她脱下鞋子，走到湖边，任由冰凉的湖水浸湿她的脚。她的心中充满了矛盾与痛苦。她想起了父亲临终前对她说的话：“阿尔娜，你是草原的女儿，你的灵魂属于这片土地，永远不要让任何人夺走你的自由。”

可如今，她连保护家人都做不到，又何谈自由？

“阿尔娜。”一个低沉而熟悉的声音从背后传来。

她猛地回头，看到耶律萧然正站在不远处。他的身影在月光下显得高大而孤独，眼神中带着一抹复杂的情绪。

“你怎么会在这里？”阿尔娜的声音中带着一丝警惕。

“我听说图尔汗要娶你。”耶律萧然走近一步，语气低沉，“这是真的？”

阿尔娜没有回答，只是冷冷地看着他。

“为什么？”耶律萧然的声音中透着一丝痛苦，“图尔汗是个贪婪而残暴的人，他不值得你——”

“够了！”阿尔娜打断他，眼中闪过一丝愤怒，“耶律萧然，你是契丹人，你根本不明白我的处境！图尔汗控制着整个部落，他威胁我的家人。如果�不答应，他会让我的母亲和弟弟死无葬身之地！”

耶律萧然沉默了。他知道阿尔娜说的是事实，但他还是无法接受。他的心中充满了矛盾。他是契丹的将领，他的职责是维护西辽的统治，而不是插手草原上的婚事。但他无法忽视内心深处那股隐隐的情感。

“如果我能帮你呢？”他低声问道。

阿尔娜冷笑，“帮我？你怎么帮？你是契丹的将领，而我是突厥的女人。我们是两个世界的人，你能帮我什么？”

耶律萧然没有回答。他知道自己的身份和职责让他无法轻易插手这件事，但他又不愿就此放弃。

“耶律萧然，回到你的军营去吧。”阿尔娜的声音低沉而疲惫，“我们的命运早已注定。你救不了我，也救不了我的家人。”

她转身离去，留下耶律萧然站在原地，目光复杂地望着她的背影。他的心中充满了无力感。他第一次意识到，自己的职责与身份不仅是荣耀，也是枷锁。

湖边的风渐渐大了起来，吹乱了阿尔娜的头发，也吹散了耶律萧然的思绪。草原的夜晚依旧宁静，但两人的心中却充满了风暴。

草原的夜，深沉而静谧。阿尔娜回到帐篷时，母亲和弟弟已经睡着了，但她却无论如何也无法入眠。她躺在毡毯上，眼睛盯着帐篷顶的横梁，脑海中一片混乱。

她知道，明天的婚礼将彻底改变她的命运，而她唯一能做的，就是接受这一切。

在湖畔的另一端，耶律萧然站在营帐外，望着远处的草原和星空。他的心中充满了挣扎。他知道自己不该插手阿尔娜的选择，但他也知道，他无法眼睁睁地看着她陷入火坑。

命运的齿轮已经开始转动，而这片草原上的风暴，似乎才刚刚开始。
